# **Segmentasi Pelanggan Menggunakan RFM & K-Means**

## **Deskripsi**
Project ini bertujuan untuk melakukan segmentasi pelanggan berdasarkan atribut RFM (Recency, Frequency, Monetary) menggunakan metode K-Means Clustering.

## **Tools & Library**
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## **Tahapan Analisis**
1. Data preprocessing
2. Perhitungan variabel RFM
3. Normalisasi data
4. Clustering menggunakan K-Means
5. Evaluasi dan visualisasi hasil

## **Output**
- Segmentasi pelanggan kedalam beberapa cluster
- Insight perilaku pelanggan berdasarkan cluster

## **Tujuan**
Membantu bisnis memahami karakteristik pelanggan untuk strategi pemasaran yang lebih efektif
